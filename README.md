2. levene의 등분산검정 : 모든 모델 비교 <br>
  $H_0$ : $s_{생산수량, 아반떼}^2 = s_{생산수량, model}^2$  for some $models$ <br>
  $H_1$ : $s_{생산수량, 아반떼}^2 \neq s_{생산수량, model}^2$ for some $models$ <br>
